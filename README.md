# run time conifig

1. Clone the repository:

   ```sh
   git clone
   ```

2. Install the dependencies (including pods):

   ```sh
   yarn bootstrap
   ```

### Running the app

1. Start the dev servers

   ```sh
   yarn start
   ```

2. Run the host-app on `ios` or `android`:

   ```sh
   yarn run:host-app:ios
   # or
   yarn run:host-app:android
   ```
